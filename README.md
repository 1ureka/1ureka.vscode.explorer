# 1ureka 的 VS Code 本地資源管理套件

本套件旨在強化 VS Code 對非程式碼資源的支援。透過新的系統級檔案瀏覽器、進階圖片元資料顯示及跨應用程式整合工具，優化從檔案管理到圖片檢視的完整工作流程，打造更直覺且安全的個人化開發環境。

---

# 本地資源的深度整合

VS Code 的工作區域與內建樹狀檔案總管雖然便於管理程式碼專案，但在處理本地資源時仍有不足。當你需要整理下載在外部的素材、瀏覽圖片資料夾的參考檔案，或是進行批量檔案操作時，往往必須切換到系統檔案總管，打斷了在 VS Code 中的工作流程。

為了實現真正的一站式管理，本擴充套件提供了一個功能完整的**系統瀏覽器**，讓 VS Code 能夠適應更多元的本地互動場景。
